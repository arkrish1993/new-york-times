# App: New York Times

This is a project demo that uses Vanilla JS, HTML and CSS to build a simple Single Page Application.


A couple of simple steps to try it out:
1. Download the zip file or clone the project into your local git environment.
2. Double-click on the index.html file.


This feature contains the following functionality:
1. It displays 'Featured' and 'Latest' News from nytimes Developer API.
2. Using the Search option, the user can filter through the 'Featured' section.
3. On clicking any of the items, a sliding window appears which shows more details about the item.
4. On scrolling, UI-side pagination has been applied to the 'Featured' section.
5. The project has been created with responsive design in mind.
